# Data

This directory holds the datasets used by `bi-cytok`. All files are loaded through
helper functions in [`bicytok/imports.py`](../bicytok/imports.py) — prefer those over
reading the files directly so that loading conventions (dropped index columns, column
renames, etc.) stay in one place.

---

## `CITEdata_SurfMarkers.parquet`

Single-cell CITE-seq surface-marker (antibody-derived) counts.

- **Shape:** 161,764 cells (rows) × 229 columns.
- **Grain:** one row per cell. Columns are surface markers/epitopes (e.g. `CD4`, `CD8`,
  `CD25`), plus a leading `Cell` integer id column that `importCITE` drops on load.
- **Values:** per-cell marker counts. `importCITE`'s docstring describes these as
  "unprocessed" counts.
- **Loaded by:** `importCITE(annotation=...)`, which pairs these counts with the
  matching annotation column from `CITE_cell_type_annotations.parquet` (aligned by row
  position).
- **Row alignment:** row `i` corresponds to the same cell as row `i` in
  `CITE_cell_type_annotations.parquet` and `RNAseqSurface.parquet`.
- **Source / citation:** Hao et al., Cell 2021,
  GSE164378, GSM5008738,	"PBMC CITE-seq ADT_3P", DOI: 10.1016/j.cell.2021.04.048

## `RNAseqSurface.parquet`

Single-cell RNA-seq expression restricted to surface-protein-coding genes.

- **Shape:** 161,764 cells (rows) × 490 columns.
- **Grain:** one row per cell. Columns are gene symbols (e.g. `TNFRSF18`, `TNFRSF4`),
  plus a leading `Unnamed: 0` index column carried over from the source.
- **Relationship to the CITE files:** contains the same cells as in.
- **Loaded by:** `importRNACITE()`.
- **Row alignment:** row `i` corresponds to the same cell as row `i` in
  `CITE_cell_type_annotations.parquet` and `CITEdata_SurfMarkers.parquet`.
- **Source / citation:** Hao et al., Cell 2021,
  GSE164378, GSM5008737, "PBMC CITE-seq RNA_3P", DOI: 10.1016/j.cell.2021.04.048

## `CITE_cell_type_annotations.parquet`

Cell type labels for the CITE-seq cells, one column per annotation scheme.

- **Shape:** 161,764 cells (rows) × 5 columns, aligned by row position to
  `CITEdata_SurfMarkers.parquet`.
- **Columns:** `CellType1`, `CellType2`, `CellType3` (weighted-nearest-neighbor (WNN)
  multimodal annotations at increasing granularity) and `CellType1_RNA`, `CellType2_RNA`
  (transcript-based annotations). `CellType2` is the default used across the analyses
  (e.g. `Treg`, `CD14 Mono`, `NK_CD56bright`).
- **Loaded by:** `importCITE(annotation="CellType2")` (returns the requested column,
  renamed to `"Cell Type"`).
- **Row alignment:** row `i` corresponds to the same cell as row `i` in
   `CITEdata_SurfMarkers.parquet` and `RNAseqSurface.parquet`.
- **Source / citation:** WNN-based annotations generated and extracted from Hao et al.,
  Cell 2021. Transcript-only annotations are generated by
  `figures/transcript-based_annotation.qmd`.

## `Sender_cell_type_densities.csv`

Literature-validated densities of immune cell types across human tissues.

- **Shape:** 257 rows × 20 columns (after dropping the leading unnamed index column
  from the source file on load).
- **Grain:** one row per `(tissue, cell_type)` pair. 47 tissues × 11 immune cell types
  (not every combination is present).
- **Loaded by:** `import_cell_densities(tissue=None)` — returns the full table, or only
  the rows for a given tissue (e.g. `"Blood"`).
- **Used in:**
  `figures/bootstrap_analyses/off-target_proportion_analysis.qmd`
  (Blood subset only), to compare literature blood cell proportions against proportions
  obtained by random sampling of the CITE-seq data.
- **Source / citation:** Sender et al., PNAS 2023, Supporting Information Dataset S02,
  DOI: 10.1073/pnas.2308511120.

### Column dictionary

The table gives, for each cell type in each tissue, a measured/estimated cell density
plus the totals it implies for three reference individuals: a 10-year-old child, an
adult woman, and an adult man. Relevant columns are described below.

| Column | Meaning |
|---|---|
| `tissue` | Tissue / organ name (e.g. `Blood`, `Liver`, `Lymph nodes`). |
| `cell_type` | Immune cell type (e.g. `B cells`, `T cells`, `Macrophages`). |
| `density` | Cell density: number of cells per unit of tissue. |
